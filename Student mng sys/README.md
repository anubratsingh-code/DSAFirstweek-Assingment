# Student Record Management System

## Objective
To manage student records using structures.

## Problem Statement
Create a program to add, display, search, update, and delete student records.

## Algorithm
1. Create a structure for student details.
2. Display the menu.
3. Perform the selected operation.
4. Repeat until Exit.

## Time Complexity
Add: O(1)
Display: O(n)
Search: O(n)
Update: O(n)
Delete: O(n)

## Space Complexity
O(n)

## Sample Input
1
101
Rahul
85

## Sample Output
Student Added Successfully!

ID   Name    Marks
101  Rahul   85.00

## Learning Outcome
Learned structures, arrays, strings, functions, and record management.
