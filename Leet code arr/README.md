# LeetCode Array Challenge

## Objective
To perform various array operations.

## Problem Statement
Find largest, second largest, smallest, second smallest, reverse the array, remove duplicates, count frequency, search, and sort the array.

## Algorithm
1. Input array.
2. Display menu.
3. Perform selected operation.
4. Repeat until Exit.

## Time Complexity
Largest/Smallest: O(n)
Reverse: O(n)
Remove Duplicates: O(n²)
Frequency Count: O(n²)
Linear Search: O(n)
Bubble Sort: O(n²)

## Space Complexity
O(n)

## Sample Input
5
4 2 7 2 9

## Sample Output
Largest = 9
Second Largest = 7
Smallest = 2
Second Smallest = 4

## Learning Outcome
Learned array manipulation, searching, sorting, and algorithm analysis.
