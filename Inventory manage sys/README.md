# Inventory Management System

## Objective
To manage product records using structures.

## Problem Statement
Create a program to add, display, search, update, sort products by price, and calculate total inventory value.

## Algorithm
1. Create a structure for product details.
2. Display the menu.
3. Perform the selected operation.
4. Repeat until Exit.

## Time Complexity
Add: O(1)
Display: O(n)
Search: O(n)
Update: O(n)
Sort: O(n²)

## Space Complexity
O(n)

## Sample Input
101
Laptop
50000
2

## Sample Output
Product Added Successfully!
Total Inventory Value = 100000.00

## Learning Outcome
Learned structures, arrays, searching, sorting, and inventory management.
