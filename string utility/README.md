Title

Assignment-02: String Utility Toolkit

Objective

To implement various string operations using arrays, loops, and functions.

Problem Statement

Develop a menu-driven program to perform:

Find string length
Reverse string
Check palindrome
Count vowels
Count digits
Count spaces
Display character frequency
Algorithm
Read a string from the user.
Display the menu.
Perform the selected string operation using functions.
Repeat until the user chooses Exit.
