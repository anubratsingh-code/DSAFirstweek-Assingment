# Dynamic Array Management

## Objective
To store student marks using dynamic memory allocation.

## Problem Statement
Create a program that dynamically allocates memory using `malloc()`, displays marks, calculates total, average, highest and lowest marks, and releases memory using `free()`.

## Algorithm
1. Read the number of students.
2. Allocate memory using `malloc()`.
3. Input student marks.
4. Display marks.
5. Calculate total, average, highest, and lowest.
6. Free the allocated memory.

## Time Complexity
O(n)

## Space Complexity
O(n)

## Sample Input
5
78 65 91 45 82

## Sample Output
Student Marks:
78 65 91 45 82

Total Marks = 361
Average Marks = 72.20
Highest Marks = 91
Lowest Marks = 45

Memory Released Successfully.

## Learning Outcome
Learned dynamic memory allocation using `malloc()`, pointers, arrays, and memory deallocation using `free()`.
