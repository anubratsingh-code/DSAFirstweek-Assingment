# Employee Payroll System

## Objective
To calculate employee salary details using structures.

## Problem Statement
Create a program to calculate HRA, DA, Tax, Gross Salary, Net Salary, display salary reports, and sort employees by salary.

## Algorithm
1. Enter employee details.
2. Calculate HRA, DA, Tax, Gross and Net Salary.
3. Display salary report.
4. Sort employees by Net Salary.

## Time Complexity
Add: O(1)
Display: O(n)
Sort: O(n²)

## Space Complexity
O(n)

## Sample Input
101
Rahul
50000

## Sample Output
Gross Salary = 65000
Net Salary = 62500

## Learning Outcome
Learned structures, arrays, functions, conditional statements, and salary calculation.
