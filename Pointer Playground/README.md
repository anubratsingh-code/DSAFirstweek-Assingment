# Pointer Playground

## Objective
To understand pointer concepts in C.

## Problem Statement
Demonstrate memory addresses, dereferencing, swapping using pointers, array traversal using pointers, and pointer arithmetic.

## Algorithm
1. Read a number and display its address and value using a pointer.
2. Swap two numbers using pointers.
3. Traverse an array using pointers.
4. Display pointer arithmetic.

## Time Complexity
O(n)

## Space Complexity
O(n)

## Sample Input
10
20 30
5
1 2 3 4 5

## Sample Output
Address of num1 = ...
Value using pointer = 10
Before Swap: 20 30
After Swap: 30 20
1 2 3 4 5

## Learning Outcome
Learned pointer declaration, dereferencing, pointer arithmetic, and passing pointers to functions.
