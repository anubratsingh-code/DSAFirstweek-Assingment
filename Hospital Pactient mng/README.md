# Hospital Patient Management

## Objective
To manage patient records using structures.

## Problem Statement
Create a program to add, display, search, update, and delete patient records.

## Algorithm
1. Add patient details.
2. Display records.
3. Search by Patient ID.
4. Update or delete records.
5. Exit.

## Time Complexity
Add: O(1)
Search: O(n)
Update: O(n)
Delete: O(n)

## Space Complexity
O(n)

## Sample Input
101
Rahul
22
Fever

## Sample Output
Patient Added Successfully.

## Learning Outcome
Learned structures, arrays, strings, and functions.
