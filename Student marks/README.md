# Student Marks Analyzer

## Objective
To store and analyze the marks of N students using arrays and functions.

## Problem Statement
Create a menu-driven program to display marks, find highest, lowest, total, average, search marks, and sort marks.

## Algorithm
1. Input number of students and marks.
2. Display the menu.
3. Perform the selected operation.
4. Repeat until Exit.

## Time Complexity
- Display/Search: O(n)
- Sort: O(n²)

## Space Complexity
O(n)

## Sample Input
5
78 65 91 45 82

## Sample Output
Highest Marks = 91
Average Marks = 72.20

## Learning Outcome
Learned arrays, functions, searching, sorting, and menu-driven programming.
